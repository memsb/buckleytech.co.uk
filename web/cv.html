Latest Projects

BBC Podcasts

BBC Podcasts is a major content distribution system that receives over 60 million unique downloads per month. Until recently it was backed by a 9 year old technology stack that was in urgent need of modernisation.

A year ago I was given the daunting task of managing the delivery of a major architectural overhaul for all BBC Podcasts. This was because the systems in place to create and publish the podcasts were around nine years old and could no longer be supported, while some parts were due to be switched off entirely. A huge amount of work and a few grey hairs later, and BBC Podcasts remain available, but on a totally new system.

Tasks

  • Renew desktop and mobile sites
  • Move all data feeds to use the new Nitro REST API
  • Migrate all 36,000 current episodes with no loss of service
  • Create new editorial management tools

Development

  • PHP using Zend Framework, API integration
  • Front-end implementation with Javascript, jQuery & CSS
  • Cross-browser multi-device automated testing using Cucumber & Browserstack
  • Release management, Build automation, Continuous integration & source control
